Q.1 = Difference between “ == “ and “ === “ operators.

In JavaScript, the "==" operator is used to compare the values of two operands

whereas the "===" operator is used to compare both the values and the data types of the operands.

== is a Strictally operater and === is a Equality operaters.

EXP.


        let a = "10";
        let b = "10";

       console.log(typeof(a));//Number
       console.log(typeof(b));//String

      console.log(a == b); 
       console.log(a === b);
